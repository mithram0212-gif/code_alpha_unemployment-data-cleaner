# 📊 Unemployment Analysis in India

## 📌 Project Overview
This project analyzes unemployment trends in India using Python. The dataset contains unemployment statistics from different states and regions across the country. The main objective is to clean the data, perform Exploratory Data Analysis (EDA), and visualize unemployment patterns to gain meaningful insights.

## 🎯 Objectives
- Load and explore the unemployment dataset.
- Clean and preprocess the data.
- Analyze unemployment rates across states and regions.
- Create visualizations to identify trends and patterns.
- Extract meaningful insights from the data.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset Information
The dataset includes:
- State
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed Population
- Estimated Labour Participation Rate (%)
- Region

## 🔍 Project Workflow

### 1. Data Loading
The dataset was imported using Pandas and inspected for structure and content.

### 2. Data Cleaning
- Checked for missing values.
- Removed unnecessary spaces and inconsistencies.
- Verified data types for analysis.

### 3. Exploratory Data Analysis (EDA)
- Analyzed unemployment rates across states.
- Compared unemployment levels between regions.
- Identified trends and variations in unemployment data.

### 4. Data Visualization
The following visualizations were created:
- Bar Chart for state-wise unemployment rates.
- Pie Chart for regional unemployment distribution.
- Line Chart for unemployment trends over time.
- Histogram for unemployment rate distribution.

## 📈 Key Insights
- Unemployment rates vary significantly across different states.
- Some regions experience higher unemployment levels than others.
- Visual analysis helps identify trends and fluctuations in unemployment.
- Data-driven insights provide a better understanding of employment patterns in India.
